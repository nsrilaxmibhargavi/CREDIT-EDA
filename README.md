## üìå Project Overview
**Credit EDA** is an interactive Exploratory Data Analysis (EDA) project built using Python to analyze customer demographics and credit card transaction data. The project focuses on understanding spending behavior, customer satisfaction, and financial patterns through structured analysis and visualizations.

All insights and plots are presented in a **single GUI dashboard**, making data exploration simple, organized, and user-friendly.

---

## üéØ Objectives
- Perform exploratory data analysis on customer and credit card datasets
- Identify trends, distributions, and relationships within the data
- Visualize insights using meaningful charts and plots
- Present results in a single interactive GUI instead of multiple pop-up windows
